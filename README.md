# 🛡️ SafeShare — Your One-Tap Safety Companion

## The Problem

Women's safety is a daily concern in India — walking home late, taking public transport alone, or navigating unfamiliar areas. In emergencies, every second counts, but existing safety apps often require subscriptions, complicated setup, or force the person in danger to fumble through menus at the worst possible moment.

## The Solution

SafeShare is a free, no-subscription safety companion built for real emergencies. One tap:
- Shares your **live location** with emergency contacts via WhatsApp — no app install needed on their end
- Shows your **device battery status** so contacts know if you might go offline
- Starts **local audio/video recording** as evidence, saved directly to your device
- Gives instant access to **police and women's helpline numbers**

Built to be simple enough to use while scared, and reliable enough to trust.

## Features

- 🚨 One-tap SOS with live location + WhatsApp alert to saved contacts
- 👥 Easy-to-manage emergency contacts list
- 🎥 On-device emergency audio/video recording
- 📞 Quick-dial police & women's helpline directory
- 🔋 Battery status shared automatically during SOS

## Coming Next (Roadmap)

HTML, CSS, JavaScript — built to run entirely in-browser, no backend required, no data leaves your device.
